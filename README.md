# Pull Before Edit VSCode Extension

Notify you when there are unpulled changes from GitHub.

## Features
- Automatically detects if remote has changes
- Shows warning notification with "Pull Now" button
- Status bar indicator

## Installation
- Installation guide: https://github.com/Kh4ht/pull-before-edit/blob/main/installation/INSTALL.md

## Usage
Just install and open any git repository. The extension will:
1. Check for remote changes every 60 seconds
3. Click "Pull Now" when prompted

## Requirements
- Git installed on your system
- VS Code 1.85.0 or newer
