# Installation Guide

This guide explains how to install and use the VS Code extension from a `.vsix` file.

---

## 📦 Install from VSIX

### 1. Install The Latest Version In This Folder
inside the installation folder download `pull-before-edit-#.#.#.vsix` with the highest number

### 2. Open VS Code
Launch **Visual Studio Code** on your machine.

### 3. Open Command Palette
Press:
For Windows: `Ctrl + Shift + P`
For MacOS: `Cmd + Shift + P`

### 4. Run Install Command
Type and select: `Extensions: Install from VSIX`

### 5. Select the Extension File
Navigate to the installed `.vsix` file and select it.

### 6. Reload VS Code
If prompted, click **Reload** to activate the extension.
